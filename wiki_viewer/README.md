# Show Local Weather - FreeCodeCamp Challenge

To run this webpage:

- Install NodeJS
- Run `npm install` in this folder
- Rename `empty-config.js` to `config.js`
  - Insert your Mapbox Api key
- Run `npm start` opens index.html in FF/Chrome
